Detailed Description

PD structured vdm command result.

it is used in PD_DPM_STRUCTURED_VDM_SUCCESS and PD_DPM_STRUCTURED_VDM_FAIL events callback. it provide vdm command reply message information to application, it may be ACK (contain data), NAK or BUSY.
